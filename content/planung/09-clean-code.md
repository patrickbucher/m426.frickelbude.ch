Im neunten Block geht es um das Thema _Clean Code_. Hierzu schauen wir uns verschiedene Qualitätskriterien für sauberen Code an. Anschliessend wenden wir das Gelernte auf ein fehlerhaftes Beispielprogramm an – welches sich nach der Codeverbesserung hoffentlich einfacher korrigieren lässt.

## Ziel

Du kannst Kriterien für sauberen Code benennen und anwenden.

## Ablauf

### Lektion 1

- [5 min.] Einstiegsbeispiel
- [5 min.] Administratives
- [35 min.] Folien zu Clean Code

### Lektion 2
